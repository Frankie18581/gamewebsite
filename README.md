# Game Website

> A retro game-themed personal website with HTML/CSS/JavaScript.

趣味向的游戏主题个人网站，纯前端实现。通过 GitHub Pages 自动部署。

## 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **部署**: GitHub Pages (GitHub Actions)
- **CI/CD**: Push → auto deploy
